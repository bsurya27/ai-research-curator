How this works

  1. L1 — The daily loop. The ten numbered actions that make up one revolution. A chase highlight walks through them in order so the flow reads like a story.
  2. L2 — Inside a connection. Click any numbered step. You'll see the two modules involved and every tool call that passes between them, stacked as parallel arrows.
  3. Arrow focus. On L2, click any main arrow: the rest of the scene dims, the side panel shows call-level detail, and you get a GitHub link to the exact line of that call in the repo.
  4. Drag to pan, scroll to zoom. The canvas is pan-and-zoomable so nothing is cut off.
